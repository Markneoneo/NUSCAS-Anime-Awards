# 🏆 NUSCAS Anime Awards

The official voting and results platform for the **NUS Comics and Animation Society (NUSCAS)** Anime Awards. This project allows club members to vote for their favorite series and showcases the final winners in an interactive format.

## 🚀 Live Site
Check out the current winners here:

## ✨ Features
* **Voting System:** User-friendly interface for selecting top anime across various categories.
* **Winners Showcase:** A dedicated gallery to celebrate the top-voted titles.
* **Responsive Design:** Optimized for both desktop and mobile viewing.
* **Static Hosting:** Fast and reliable deployment via GitHub Pages.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** Google Sheets, Google Apps Script
* **Hosting:** GitHub Pages
